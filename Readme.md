# Review&RATE

Live URL: https://review-rate-mu.vercel.app/

Review&RATE is a MERN stack application that allows users to add companies, submit reviews, view company listings, and explore detailed review statistics. The UI is built according to the provided Figma prototype and focuses on clean design, efficient functionality, and a smooth user experience.

## Features

### Company Management

- Add new companies with details such as name, city, location, and founded year
- View updated company listings with search and filter functionality
- Company profile page displaying logo, description, and essential details

### Reviews

- Add reviews for a company with reviewer name, subject, message, and rating
- View review lists with sorting options
- Automatic average rating calculation
- Interactive review actions

## Tech Stack

### Frontend

- React
- TailwindCSS
- React Router
- Lucide Icons

### Backend

- Node.js
- Express.js
- MongoDB + Mongoose
- Cloudinary

## Deployment

- Frontend deployed on Vercel
- Backend deployed on Render
